## 类型转换函数

| 函数语法                | 描述      | 示例                                       | 返回值              |
| ------------------- | ------- | ---------------------------------------- | ---------------- |
| CAST(value AS type) | 转换成值的类型 | ```select cast(CURRENT_DATE as varchar )``` | ```2017-08-10``` |

注：此函数不适用于可计算列。有关可计算列，参见**数据建模**一章中的[可计算列](model/computed_column.cn.md)一节。
